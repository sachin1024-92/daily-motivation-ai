# Daily Motivation AI 🚀

**Grok-powered daily motivation, habit tracking & focus app** built with Flutter.

Turn every day into a productive, positive one with AI encouragement, smart reminders, and beautiful minimal design.

![App Preview](https://via.placeholder.com/800x500?text=Add+your+app+screenshots+here)

## Why Daily Motivation AI?

Most motivation apps are static. This one is **alive**:
- Personalized AI coach (powered by Grok) that adapts to your mood and goals
- Smart habit streaks + gentle nudges
- Focus timers with motivational voice notes
- Daily quotes + reflections tailored to you
- Completely free core experience + optional premium features

Perfect for students, professionals, and anyone building better daily habits while preparing for big goals (like government exams or side hustles).

## Key Features

- ✅ AI-powered daily motivation & personalized encouragement
- ✅ Habit tracker with smart streaks and reminders
- ✅ Pomodoro + deep focus timers
- ✅ Mood check-ins + AI insights
- ✅ Beautiful, calm, minimal UI (dark/light mode)
- ✅ Export progress & share achievements
- ✅ Offline support

## Tech Stack

- **Flutter 3.x** + Dart
- **Firebase** (Auth, Firestore, Analytics)
- **Grok / xAI API** integration for smart responses
- Local notifications & widgets
- Clean architecture + Riverpod / Bloc state management

## Getting Started

```bash
git clone https://github.com/sachin1024-92/daily-motivation-ai.git
cd daily-motivation-ai
flutter pub get
flutter run
```

Add your Firebase config and Grok API key in the appropriate files.
